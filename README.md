# Eksperimen Operasi Dasar pada Sinyal dan Citra

**Nama:** Brucad Al Magribi  
**Mata Kuliah:** Pengolahan Sinyal Digital  


## Deskripsi Singkat
Repositori ini berisi implementasi eksperimental mengenai operasi dasar pengolahan sinyal 1D diskrit dan citra digital 2D menggunakan bahasa pemrograman Python. Project ini berfokus pada pembuktian matematis sifat linieritas sistem (homogenitas dan aditivitas) serta analisis visual efek modifikasi sinyal pada skenario dunia nyata.

## Library Utama yang Digunakan
* **NumPy:** Pemrosesan array multi-dimensi dan komputasi matriks numerik cepat.
* **Matplotlib:** Visualisasi grafik isyarat diskrit dan analisis distribusi histogram citra.
* **OpenCV (cv2):** Manipulasi transformasi affine gambar spasial dan manajemen tipe data citra digital.
